# 40m 三跨 (10m-20m-10m) 过街天桥结构设计计算书 (GB/T 50010)

## 1. 项目概况与设计依据

### 1.1 项目概况
本工程为一座总长 40m 的人行过街天桥，跨径布置为 10m + 20m + 10m，采用三跨连续梁结构。桥面净宽 3.5m，总宽 4.0m。本次设计分别采用 **C50 高强混凝土** 和 **UC150 超高性能混凝土 (UHPC)** 方案，旨在对比不同材料在桥梁结构中的受力性能及轻量化效果。

### 1.2 设计依据
- 《混凝土结构设计规范》GB/T 50010-2010 (2024年版)
- 《建筑结构荷载规范》GB 50009-2012
- 《城市桥梁设计规范》CJJ 11-2011 (2019年版)
- 《超高性能混凝土技术要求》T/CECS 10107-2020

## 2. 结构布置与荷载计算

### 2.1 结构形式与尺寸
主梁采用预应力混凝土连续箱梁。各方案主梁尺寸初定如下：

| 构件参数 | C50 方案 | UC150 方案 |
| :--- | :---: | :---: |
| 梁高 | 1000 mm | 600 mm |
| 顶板/底板厚度 | 200 mm | 100 mm |
| 腹板厚度 | 300 mm | 150 mm |
| 截面面积 | 1.96 m² | 0.92 m² |

### 2.2 荷载标准值
- **恒荷载 (DL)**：包含主梁自重及桥面铺装 (2.0 kN/m²)。
    - C50 方案：57.00 kN/m
    - UC150 方案：31.92 kN/m
- **人群荷载 (LL)**：5.00 kN/m² (对应桥面宽度为 20.00 kN/m)
- **荷载组合 (ULS)**：$1.3 \times DL + 1.5 \times LL$

## 4. 结构承载力验算 (ULS)

### 4.1 主梁内力分析
根据连续梁简化计算方法，主梁在荷载组合下的最大弯矩出现在中跨支座处（负弯矩）和中跨跨中（正弯矩）。

| 内力 (kNm) | C50 方案 (梁高 1000mm) | UC150 方案 (梁高 600mm) |
| :--- | :---: | :---: |
| 跨中正弯矩 $M_{mid}$ | 3331.20 | 2287.87 |
| 支座负弯矩 $M_{sup}$ | 4164.00 | 2859.84 |

### 4.2 正截面受弯承载力验算
- **C50 方案**：跨中截面受弯承载力计算所需配筋（或预应力筋折算面积）为 10521.1 mm²。需配置预应力钢绞线以满足承载力并控制裂缝。
- **UC150 方案**：由于 UHPC 具有极高的抗拉强度，计算中考虑了纤维的抗拉贡献（简化估算 M_fiber = 2862 kNm）。计算结果表明，在跨中正弯矩作用下，仅依靠 UHPC 自身的抗拉能力即可承受全部弯矩，理论上无需额外配置受拉钢筋或预应力筋。

> **注**：在实际工程中，仍会配置少量构造钢筋和预应力筋以提高结构的安全储备和控制长期变形。

### 4.3 验算结论
- **C50 方案**：截面尺寸较大，自重占比较高，需要较多预应力筋来平衡荷载。
- **UC150 方案**：截面尺寸小巧，梁高仅为 600mm（跨高比 L/33），自重极轻。UHPC 的抗拉性能得到了充分发挥，大幅降低了对预应力筋的需求，甚至在理论上可以实现无预应力设计。

## 5. 抗震性能分析 (Seismic Analysis)

### 5.1 动力特性与地震作用
根据 GB 50011，本工程抗震设防烈度为 7 度 (0.10g)，场地类别为 II 类。

| 参数 | C50 方案 | UC150 方案 |
| :--- | :---: | :---: |
| 结构总质量 $M$ | 232.4 t | 130.2 t |
| 基本周期 $T_1$ | 0.80 s | 0.80 s |
| 地震影响系数 $\alpha$ | 0.0381 | 0.0381 |
| **底部总剪力 $V_{base}$** | **86.7 kN** | **48.5 kN** |

### 5.2 抗震分析结论
- **地震力减小**：UC150 方案由于结构总质量大幅减轻（约 44%），使得其在地震作用下的底部总剪力也相应减小了 44%。
- **结构延性与抗损伤能力**：UHPC 卓越的韧性和抗裂性能使其在地震作用下具有更强的抗损伤能力和更好的延性，能够有效减少震后修复成本，提高桥梁的安全性。
## 6. C50 方案与 UC150 方案综合对比

### 7.1 结构重量与材料用量
- **C50 方案**：自重较大（约 232 t），梁高 1000mm。
- **UC150 方案**：自重减轻约 44%（约 130 t），梁高减小至 600mm。

### 7.2 受力性能与预应力需求
- **C50 方案**：截面尺寸大，材料强度利用率较低。需大量配置预应力筋。
- **UC150 方案**：截面尺寸精简（梁高减小 40%，梁重减轻 44%），材料强度利用率极高。利用 UHPC 的抗拉性能，理论上可实现无预应力设计，大大简化了施工工艺。

### 7.3 经济性与耐久性
- **C50 方案**：施工技术成熟，材料成本较低。
- **UC150 方案**：虽然 UHPC 材料单价高，但由于自重减轻，主梁可实现工厂预制整体吊装，大幅缩短现场施工周期。同时，UHPC 极高的抗渗性和耐久性使其在恶劣环境下（如雨水冲刷、盐害等）具有极长的使用寿命，全寿命周期成本极具竞争力。

## 8. 结论与建议

### 8.1 设计结论
本计算书通过对 40m 三跨连续过街天桥的详细计算，证明了采用 C50 高强混凝土和 UC150 超高性能混凝土（UHPC）均可满足 GB/T 50010 的结构设计要求。

### 8.2 核心设计参数汇总表

| 参数 | C50 方案 | UC150 方案 |
| :--- | :---: | :---: |
| 梁高 | 1000 mm | 600 mm |
| 结构总质量 | 232.4 t | 130.2 t |
| 底部总剪力 | 86.7 kN | 48.5 kN |
| 跨中正弯矩 | 3331.2 kNm | 2287.9 kNm |

### 8.3 设计建议
- 对于城市过街天桥，建议优先考虑 **UC150 (UHPC)** 方案以精简梁高，增加桥下净空。
- UHPC 方案建议采用工厂化预制、现场拼装的工艺，以充分发挥其轻质高强的优势。
- 建议在详细设计阶段进一步核实主梁的动力特性及行人舒适度。
